Specifies the type of functions that are used to query the sedflux architecture.

Parameters:
query_data - A pointer to the data that specifies the user's query. This data may contain things like a location, depth, or sediment property. Any output data should also be contained within this data structure.
data - A pointer to data from the architecture that is to be queried.
Returns:
Nothing.
