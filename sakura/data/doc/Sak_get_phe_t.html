Documentation

The Sak_get_phe_t structure describes the grain size distribution of the bottom sediments used in the sakura turbidity current model. It is intended to be passed as the data parameter of a Eh_query_func.

The phe_bottom member gives the fraction of each grain type that is found on the sea floor.

The n_grains member is the number of grain types (the length of phe_bottom ).
